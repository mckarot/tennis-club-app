# PRD: Application Tennis Club du François (Martinique) ## Objectif Créer une Progressive Web App (PWA) de gestion et réservation pour le tennis club municipal du François, Martinique. ## Stack Technique - **Frontend:** React 19+ (Server Components), Tailwind CSS 4.0+ - **Backend/Auth:** Firebase Suite (Auth, Firestore, Storage) - **State Management:** TanStack Query - **Date/Time:** Day.js (Fuseau: America/Martinique) ## Schéma de Données (Firestore) - `users`: { uid, name, email, role: admin|moniteur|client, phone } - `courts`: { id, name, type: Quick/Terre, is_active } - `reservations`: { id, court_id, user_id, start_time, end_time, type, status } - `slots_moniteurs`: { id, moniteur_id, start_time, end_time, available } ## Design & UI - **Thème:** Professionnel, sportif, épuré. - **Palette:** - Primaire: Vert Court (extrait de l'image {{DATA:IMAGE:IMAGE_1}}) - Secondaire: Ocre (Terre battue) - **INTERDICTION:** Pas de rose. - **Composants clés:** Cards pour les cours, Time-grid interactive pour les réservations. ## Flux Utilisateurs 1. **Client:** Réservation de court (1h), vue calendrier filtrable. 2. **Moniteur:** Gestion d'agenda, définition de créneaux, liste de participants. 3. **Admin:** Supervision, blocage de courts (maintenance/événements), gestion utilisateurs.
